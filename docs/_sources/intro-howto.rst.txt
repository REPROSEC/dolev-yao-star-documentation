How To: Prepare Protocol Models for DY*
---------------------------------------

.. howto:: Prepare Protocol Model for DY*

   1. Reformulate the intuitive description of the protocol to
      a local description using *receive-react steps*
   2. For each of the receive-reat steps think about:

      * Who is the active participant?
      * What are the *actions* performed

        * What messages are sent?
        * Is there a change in state?
      * What *information* has to be stored in the states of participants
        for them to continue the next steps?
